What is jikAPI?

jikAPI (Jobs In Kenya Live API) is a Job Search API that allows users to search for job opportunities that have been recently posted by employers in Kenya. It will provide a simple and easy-to-use interface for users to find the perfect job for them returning 400+ unique available jobs in Kenya posted within the last 7 days.
This is the easiest way to get access to the latest job postings and openings in Kenya, link to the job application page and the date posted.

Accuracy

jikAPI has a mechanism in place to regularly update and verify the job listings it provides to ensure that they are up-to-date and accurate.

Inclusiveness

jikAPI offers a diverse range of job listings from different sources, sectors and industries, making it a one-stop shop for job seekers in Kenya.

Integration

jikAPI provides content that will be easily embedded or interwoven throughout sites or other applications ensuring smooth and integrated user experience.

Coming soon...

The API will soon include customizable search filters, a robust security system to protect user's data and ensure that the API is always available and responsive, provide more detailed information about each job listing than other similar API's, such as salary range, company culture, benefits, etc.
